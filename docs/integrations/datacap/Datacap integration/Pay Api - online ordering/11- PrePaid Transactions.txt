	Set
The Set command is used to modify Prepaid account attributes without affecting the account balance. It provides functionality to manage custom identifiers, account locks, and credit limits.
POST /storedvalue/set
Implementation Notes:
1. The Set field can only be used with Datacap Gift™
2. CreditLimit cannot be set on an account that does not have a custom identifier.
3. Set must specify at least one of: NewIdentifier, Locked, or CreditLimit. NewIdentifer must must be unique within your business.
4. Set is the only request that works on locked accounts.
5. NewIdentifier set to empty string (“”) removes the custom identifier and removes any associated CreditLimit.
