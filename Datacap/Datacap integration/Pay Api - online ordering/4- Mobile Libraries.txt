Mobile Libraries
Datacap’s one-time-use token libraries are also availalbe in iOS and Android mobile formats. These include prebuilt user interfaces to manually collect card data. Just drop in the form, enter your authentication information, and you’re ready to receive tokens in your mobile application.

A mobile tokenization solution using an iOS library.
